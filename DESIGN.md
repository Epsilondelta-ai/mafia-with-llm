# 게임 기획서 v1.0 — 마피아 카드게임

## 1. 개요

턴제 숨겨진 역할 카드게임. 사람과 AI(코드 AI / LLM AI)가 함께 플레이.

### 테마
마피아. 도시에서 시민과 경찰이 마피아를 찾아내야 한다.

### 핵심 차별점
- LLM AI는 자연어 채팅으로 설득/거짓말/추론 가능
- 코드 AI는 규칙 기반 행동 + 기계적 템플릿 발언
- 무료: 코드 AI / 유료: LLM AI

---

## 2. 역할

### 진영
- **경찰** — 역할 공개. 체력 +1. 정체가 공개된 마피아만 공격 가능.
- **시민** — 역할 비공개.
- **마피아** — 역할 비공개. 마피아끼리는 서로 인지.

### 승리 조건
- **시민 진영 (경찰+시민):** 마피아 전원 제거.
- **마피아 진영:** 경찰 제거.
- **시민이 경찰을 제거한 경우:** 마피아 승리.

### 탈락
- 탈락 시 역할 공개.
- 구급차가 손패에 있으면 자동 발동, 체력 1로 부활. 구급차만 소멸, 나머지 손패 유지.
- 구급차가 없으면 영구 사망. 모든 카드 버림.

### 인원별 역할 분배

| 인원 | 경찰 | 시민 | 마피아 |
|------|------|------|--------|
| 5인 | 1 | 2 | 2 |
| 6인 | 1 | 3 | 2 |
| 7인 | 1 | 3 | 3 |
| 8인 | 1 | 4 | 3 |

### 마피아 정체 공개
- 자기 턴 카드 사용 단계에서 자발적으로 역할 공개 가능. 되돌리기 불가.
- **공개 즉시:** 카드 2장 추가 드로.
- **공개 전:** 킬 보상 없음 (마피아 제거 시에는 보상 있음).
- **공개 후:** 누구든 제거 시 체력 1회복 + 카드 2장 드로.

## 3. 플레이어 구성

- **체력** — 기본 3. 경찰은 4.
- **역할** — 경찰 / 시민 / 마피아

---

## 4. 턴 구조

### 턴 순서
경찰이 첫 순서. 순차진행.

### 턴 흐름
1. **채팅** — 택1:
   - **공개 발언:** 전체에게 한 마디. 대답 없음.
   - **1:1 질문:** 한 명 지목. 질문과 대답 모두 전체 공개. 대답 한 문장 제한. 거부 가능.
2. **카드 뽑기** — 2장. 공격 카드 뽑으면 즉시 전체 공개.
3. **카드 사용** — 제한 없음. (마피아는 이 단계에서 정체 공개 가능)

### 제한시간
- 각 행동(채팅/카드뽑기/카드사용) 후 **1분(60초)** 타이머가 리셋된다.
- 전체 턴이 아닌 **개별 행동 단위**로 제한시간이 적용된다.
- 제한시간 내에 행동하지 않으면 **현재 단계만** 자동 진행된다:
  - **채팅 단계:** 발언 건너뛰기 → 타이머 리셋
  - **카드 뽑기 단계:** 자동 드로 → 타이머 리셋
  - **카드 사용 단계:** 턴 종료
- AI 플레이어에게는 제한시간이 적용되지 않는다.

### 기타
- 발언 기록은 전체 로그로 누적.

---

## 5. 카드 시스템

### 정보 공개
- **공개:** 공격 카드 보유 현황
- **비공개:** 회복/방해/구급차
- **⚠️ 중요:** 공격 카드는 덱에서 랜덤 드로이므로, 총 보유량은 역할(마피아 여부)과 완전히 무관함. 총을 많이 가진 것은 의심 근거가 아님.

### 시작 손패
1장. 손패 상한 없음.

### 덱 소진
버린 카드를 셔플하여 재활용 (구급차는 사용 시 영구 제거).

## 6. 카드 목록

### 공격 (공개)

| 테마명 | 효과 | 수량 |
|--------|------|------|
| 총격 (SHOT) | 대상 1명에게 1데미지 | 40장 |
| 저격 (SNIPE) | 대상 1명에게 2데미지. 총격과 혼용 불가. 사용 후 추가 공격 불가. 다음 턴 드로 불가 | 6장 |

### 회복 (비공개)

| 테마명 | 효과 | 수량 |
|--------|------|------|
| 응급처치 (FIRST AID) | 체력 1 회복 | 10장 |
| 입원 (HOSPITAL) | 체력 2 회복. 해당 턴 공격 불가 | 4장 |

### 방해 (비공개)

| 테마명 | 효과 | 수량 |
|--------|------|------|
| 체포 (ARREST) | 대상의 다음 턴 공격 불가 (턴 종료 시 해제) | 6장 |
| 압수 (SEIZE) | 대상의 공개된 공격 카드 1장 지정 파괴 | 7장 |
| 수색 (SEARCH) | 대상의 비공개 카드 1장 랜덤 파괴 | 5장 |

### 특수 (비공개, 자동 발동)

| 테마명 | 효과 | 수량 |
|--------|------|------|
| 구급차 (AMBULANCE) | 사망 시 자동 발동. 체력 1로 부활. 구급차만 소멸. 영구 제거. **부활 후 공격 패널티 없음.** | 5인: 2장 / 6~8인: 3장 |

### 기본 덱: 90장 + 구급차 (공격 46, 회복 14, 방해 18)

## 7. 벌칙 / 보상

| 상황 | 결과 |
|------|------|
| 경찰의 공격 대상 | 정체가 공개된 마피아만 가능 |
| 경찰에 대한 공격 | 정체를 공개한 마피아만 가능 |
| 경찰에 대한 방해 카드 사용 (체포/압수/수색) | 정체를 공개한 마피아만 가능 |
| 경찰, 시민, 정체를 공개하지 않는 마피아가 시민을 죽일 시 | 체포(ARREST) 상태가 됨 |
| 마피아 정체 공개 시 | 즉시 카드 2장 추가 드로 |
| 누구든 마피아 제거 | 체력 1회복 + 카드 2장 드로 |
| 마피아(공개)가 누구든 제거 | 체력 1회복 + 카드 2장 드로 |
| 마피아(비공개)가 누구든 제거 | 보상 없음 |
| 저격 사용 | 그 턴 총격과 혼용 불가. 사용 후 추가 공격 불가 |

## 8. AI 차별화

### 코드 AI (무료)
- 확률 기반 행동 선택
- 기계적 템플릿 발언 ("N번 유저의 위협 수치가 높습니다")
- 패턴이 읽히므로 숙련 플레이어는 예측 가능
- 마피아 역할 시 전략적 정체 공개: 공격카드 2장 이상, 경찰 체력 낮을 때, 후반전(3명 이하) 등 상황에 따라 정체 공개
- **정체 공개된 마피아 인지:** 90% 확률로 정체 공개된 마피아를 우선 타겟팅. 채팅에서 협력 공격 유도 발언.

### LLM AI (유료)
- 전체 게임 맥락을 종합하여 판단
- 자연어 채팅으로 설득, 거짓말, 모순 지적 가능
- 발언이 자연스럽고 예측이 어려움
- 마피아 역할 시 전략적 판단으로 정체 공개 여부 결정 (공격카드 보유량, 경찰 체력, 게임 상황 고려)
- **정체 공개된 마피아 인지:** 프롬프트에 정체 공개된 마피아를 명시적으로 알림. 우선 공격 및 채팅에서 협력 유도.
